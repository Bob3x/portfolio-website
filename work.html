pokedeXx

Overview

Pokedex is simple web application built with HTML, CSS and JavaScript. Users can browse through a list of Pokemon cards and select one by name. By clicking on a Pokemon image and more details about the character are displayed. Friendly and simple UI with individual modal providing more information. The app is fetching data from the official Pokemon API filtered to selected elements.

meeet App

Overview

meeet is a React application, developed with TDD and BDD approach, using Serverless framework deployed to AWS Lambda. It's modern and scalable PWA with cool features. Users can browse events available in selected City. Event's data is fetched from Google Calendar API. meeet App can be installed on smart phones and used offline as well. Last but not least colorful charts are visualizing events information.
